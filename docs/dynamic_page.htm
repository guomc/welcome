一、简介

1.php

主要用于数据库的读写，将h5的文件名后缀变更为“.php”、在需要的地方嵌入“<?php…?>”代码即可，比如：

<?php

       //“echo 值;”在页面中输出内容、这个值可以是数或字符串，而且在字符串中的HTML会被识别；通过浏览器看不到注释的内容

echo "我的第一段 PHP 脚本！";

?>

2.环境设置

安装并启动xampp；

将h5或php文件放在“…/xampp/htdocs/工程目录”下；

通过“http:80//工程目录/…”访问即可

 

二、数据类型

“print_r(对象)”可以输出对象的类型和内容；布尔类型true、false，用var定义不需要明确的类型

1.数和字符串

最好用“===”做比较、不做类型转换，布尔值TRUE和FALSE可以比较但不要直接输出；字符串连接用“.”，双引号中的变量会被解析

strlen(字符串对象)

返回该字符串的长度

str_word_count(字符串对象);

返回字符串中的单词个数

strrev(字符串对象)

返回反向的字符串

strpos(字符串对象0,字符串对象1)

字符串0中是否有字符串1，

如果有返回第一次出现的下标，

否则返回FALSE

str_replace(字符串对象0, 字符串对象1, 字符串对象2)

在字符串2中将字符串0替换成字符串1，

替换成功返回替换后的字符串，

否则返回原字符串

2.常量和变量

创建一个全局常量

define("常量名称", "值");

创建一个变量，变量名称由“$”开头；和其他语言类似，变量有不同的作用域

变量名称=值;

static 变量名称=值

4.数组

嵌套可以得到多维数组、通过对应的方式访问想要的值

$x=array(对象,…,对象);

或者

$x=array();

$x[0]=0;

$x[1]=1;

…;

获取数组的长度

count(数组对象)

定义一个映射数组，使用“字符串”作下标访问数组中的值

arrary(字符串=>对象,…,字符串=>对象)

遍历一个映射数组

foreach(映射数组对象 as 作为键的变量=>作为值的变量)…

5.时间日期

时间日期中的关键字依次表示“年”、“月”、“日”、“星期几”、“时”、“分”、“秒”、“上下午”，关键字可以分开使用

//不设置默认是格林尼治时区

date_default_timezone_set("Asia/Shanghai");

//第二个参数是时间戳，默认是当前时间日期

$obj=date(“Y字符串m字符串d字符串l字符串h字符串i字符串s字符串a”);

echo $obj;

四、页面交互

1.Cookie

增添Cookie

//失效时间的单位是秒

setcookie(“键”,”值”,time()+数字);

删除/修改Cookie

//相当于修改，只是修改了失效时间

setcookie(“键”,””,time-数字);

获取

//获取某个cookie

$_COOKIE[“键”];

//获取所有cookie

print_r($_COOKIE);

2.Session

会话Session存储临时的信息，可以在同一网站的多个页面中使用，当用户离开网站时Session被销毁

//开始会话，每一个在Session中的页面都需要这条语句

session_start();

//设置Session中的变量

$_SESSION[“键”]=值;

//获取Session中的变量

echo $_SESSION[“键”];

//判断Session中的某个变量是否存在

if(isset($_SESSION[“键”]))是;否;

//销毁Session中的某个变量

unset($_SESSION[“键”]);

//销毁整个Session

session_destroy();

3.表单

$_GET[“name作为键”]、$_POST[“name作为键”]获取表单中有“name”属性的输入框的值，可能用到的函数；js和php都可以做表单验证，php的做法是把action的地方编入代码，验证正确返回应该提交的页面、否则继续提交给当前页面

empty($_POST[‘name作为键’])

$_POST或$_GET的值是否为空的判断方法，不为空、返回值才有意义

if($_SERVER['REQUEST_METHOD']=='POST'){…}

POST提交后某些代码才会起效

trim(字符串)

去掉字符串两边的空格并返回

htmlspecialchars(字符串)

转义为普通字符串后返回

htmlspecialchars($_SERVER["PHP_SELF"])

当前页面的位置

header("location:./form.php")

定位到某个页面

exit()

到此为止，不执行后面的代码

preg_match(“参考js的正则表达式”,字符串)

字符串是否匹配某个正则表达式

五、数据读写

1.文件

r只读，w只写，a追加，x只写未存在的文件、否则返回FALSE

(1)读文件

读取结果以换行、空格为分隔符

一次性读取

<?php

$myfile = fopen("webdictionary.txt", "r") or die("Unable to open file!");

$cnt=fread($myfile,filesize("webdictionary.txt"));

echo "内容是：<br>".$cnt;

fclose($myfile);

?>

逐行读取直到结束

<?php

$myfile = fopen("webdictionary.txt", "r") or die("Unable to open file!");

// 输出单行直到 end-of-file

while(!feof($myfile)) {

  echo fgets($myfile)."<br>";

}

fclose($myfile);

?>

逐字读取直到结束

<?php

$myfile = fopen("webdictionary.txt", "r") or die("Unable to open file!");

// 输出单字符直到 end-of-file

while(!feof($myfile)) {

  echo fgetc($myfile);

}

fclose($myfile);

?>
